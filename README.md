# Steps

- Create virtual environment

    ```
        mkdir venv
        cd venv
        virtualenv .
        source bin/activate
    ```

- Create flask folder 

    ```
        mkdir flaskr
        cd flaskr
    ```

`Note: I did not use * flask init *`

- Create **__ init __.py** and **app.py**, write the code.

- Run flask app

    ```
        flask run
    ```
